# Source Code Projection

Repository ini bukan _source code_ utama dari project IPB Space.

Struktur folder pada repository ini dibuat khusus untuk memenuhi kebutuhan pengelompokan source code berdasarkan kategori keamanan informasi pada tugas KOM1315.

Source code asli dengan struktur project yang sebenarnya dapat dilihat pada repository berikut:

[IPB Space Main Repository](https://github.com/sam-is-grinding/ipb-space)

Repository ini hanya berisi hasil pemetaan sebagian source code dari repository utama ke dalam folder:
- `backend/` # Implementasi modul kriptografi (encrypt/decrypt)
- `database/` # Skema DB & implementasi Hashing
- `digital_signature/` # Fitur autentikasi & non-repudiation

untuk mempermudah proses peninjauan aspek keamanan informasi sesuai kebutuhan tugas.
